Preferences

You can change the keyboard's default language by adjusting kblang=\'English\''; in the the bookmark, e.g. kblang=\'Русский\'';

Technical information

The bookmarklet calls keyboard.css and kb.js, kb-deadkeys.js or kb-slav.js/keyboard-ees.js respectively and applies it to the current web site. Furthermore the JS calls the image file keyboard.png from the same directory. Consequently, the bookmarklets will only work with an established internet connection.

The bookmarklets may be downloaded from GitHub.
