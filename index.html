Bookmarklets for a multilingual virtual online keyboard

The bookmarklet displays a virtual keyboard interface in almost any web site. Works on input and textarea fields.

for East European studies DEMO

Preferences

You can change the keyboard's default language by adjusting kblang=\'English\''; in the the bookmark, e.g. kblang=\'Русский\'';

Technical information

The bookmarklet calls the JavaScript kb.js / kb-slav.js / keyboard-ees.js and the respective CSS-stylesheet keyboard.css / kb-slav.css / keyboard-ees.css and applies it to the current web site. Consequently, the bookmarklets will only work with an established internet connection.

The bookmarklets may be downloaded from GitHub.

Credits

The bookmarklet is based on the Virtual keyboard interface script by GreyWyvern.
